# `assemble_tools/bionano` — BioNano optical-map helpers

Small helpers for **BioNano** optical-map scaffolding output (`.xmap`).

| script | role |
|--------|------|
| `rename_xmap.pl` | rename contig/map IDs in a raw `.xmap` (`raw.xmap new.xmap`) |
| `sv_related_xmap.pl` | pull SV-related alignments from an `.xmap` by confidence/coverage (`-minConfidence`, `-maxCov`, …) |

_Hand-written._
